# Algorithm Simulations

Interactive visualizations of machine learning and data structure algorithms built with React, TypeScript, and Vite.

## Available Simulations

- **K-Means Clustering**: Standard K-Means with step-by-step execution and centroid movement visualization.
- **K-Means++ Initialization**: Improved seeding algorithm with side-by-side comparison against random initialization.
- **K-Nearest Neighbors (KNN) Classification**: Interactive comparison of 1-NN, K-NN, and n-NN algorithms with draggable query points.

## Features

- Highly interactive visualizations
- Step-by-step algorithm execution
- Configurable parameters
- Real-time updates
- Clean, responsive UI with shadcn/ui components

## Getting Started

```bash
npm install
npm run dev
```

Navigate to the simulations from the home page.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Lucide Icons
